JCSUnity - Save System 儲存機制

July 3, 2018 Unity 本文总阅读量次

This is a introduction of how JCSUnity does for the game data save/load system.

這是一個簡易介紹給予展示 JCSUnity 是如何做到遊戲儲存機制.

There are main three ways that you could do saving and loading in Unity Engine with C#.

在Unity Engine裡面, 有三個主要方式做遊戲資料儲存使用C#語言.

  • JCS_BinGameData
    • In binary format. 二進制格式
  • JCS_XMLGameData
    • In XML format. XML格式
  • JCS_JSONGameData
    • In JSON format. JSON格式
